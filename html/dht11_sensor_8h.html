void dht11Read ( SharedState & state)

Read temperature + humidity and write results into SharedState.

On success:

  • state.tempC, state.humidityPct updated
  • state.hasDht = true

On failure:

  • tempC/humidityPct set to NAN
  • state.hasDht = false
